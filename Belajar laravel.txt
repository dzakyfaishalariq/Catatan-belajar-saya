Belajar Larafel
->membuat file laravelnya:
--> composer create-project laravel/laravel namafolder-laravel
->menjalankan server:
-->php artisan serve

cara ke dua
->instal laravelnya:
-->composer global require laravel/installer
->buat file applikasinya :
-->laravel new example-app
->menjalankan server:
-->php artisan serve

pembelajaran :
1. fokus ke routers ke file web.php
Route::get('/', function () {
    return view('welcome'); // nama file ditulis di antara tanda petik
});
2. jangan lupa untuk membuat file di viwes gunakan namafile.blade.php
  -> kegunaan blade untuk melakukan templating 	enjing dalam laravel.

3.menggunakan sintaks blade
	-> {{$variabel}}
4. megunakan templating blade
	1. extends('namafolder.app')
	2. @yielid('namaconten') // diletakan di 			leyouts app nya
	3. @section('contentSaya')
		....conten....
	   @endsection
	4. @include('partials.navbar') // digunakan 		untuk menggunakan atribut yang dipisah 		dari lyouts.main
	5.  {{ ($title === "Blog")? 'active' : '' }}
		//untuk melakukan pengerakan tombol 			active dari bootstrep
	6. @add()//digunakan untuk melihat dom and 			day untuk melihat isi variabel mirip 			var_dam
	7. membuat parameter ling 						/blog/{{$post['']}}
	8. menggunakan perulangan di blade
		@foreach($posts as $post)
		@endforeach 
5. model dalam laravel
	membuat terlebih dahulu
	$ php artisan make:model Post
	$ php artisan make:model namefile -mfs
	yang mana -mfs itu  adalah 
	{m -> melakukan migrasi
	 f -> membuat file factorynya
	 s -> membuat sider nya}
6. controler dalam laravel
	membuat terlebih dahulu
	$php artisan make:controller namacontroller 
7. membuat dan menyambungkan ke database
	-membuat tabel:
		php artisan make:migration nameTabel
		atau
		php artisan migrate
		ingat ubahnya di file .env
	-menghapus semua tabel
		 php artisan migrate:rollback
	-melakukan keduanya sekaligus menghapus lalu membuat semuanya lagi
		php artisan migrate:fresh(recomendet)
		php artisan migrate:fresh --seed (sekalian memasukan databaseSeed nya)
8. melakukan interaksi terhadap database
	php artisan tinker

Schema::create('photos', function (Blueprint $table) {
            $table->id();
            $table->unsignedBigInteger('album_id')->nullable();
            $table->unsignedBigInteger('location_id')->nullable();
            $table->foreignId('member_id')->constrained('members')->onDelete('cascade');

            $table->string('judul', 120);
            $table->string('penjelasan')->nullable();
            $table->string('pribadi', 20);
            $table->date('tanggal_unggah');
            $table->date('tanggal_unggah');
            $table->integer('view')->default(0);

            $table->string('nama_file');

            $table->foreign('album_id')->references('id')->on('albums')->onDelete('cascade');
            $table->foreign('location_id')->references('id')->on('locations')->onDelete('cascade');
        });
